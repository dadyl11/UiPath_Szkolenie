Moduł 1 — Data Scraping
Ćwiczenie 1 — Data Scraping: CEIDG
Folder: Dzien2_Zadanie1_DataScraping_CEIDG/ · Temat: Scraping danych + filtrowanie + sortowanie + eksport

Opis zadania

  1. Wejdź na stronę CEIDG: aplikacja.ceidg.gov.pl/ceidg.public.ui/search.aspx
  2. Wpisz "Wrocław" w pole Miejscowość i "Hubska" w pole Ulica
  3. Kliknij przycisk "Szukaj"
  4. Pobierz max 50 wpisów z tabeli wyników (kolumny: Firma, Przedsiebiorca, NIP, REGON, Informacje)
  5. Usuń wpisy, w których brakuje REGON lub NIP — Filter Data Table w trybie Remove
  6. Posortuj firmy po kolumnie "Firma" (Ascending)
  7. Zapisz wynik do pliku CEIDG_Tabela_Filtr.xlsx (arkusz: CEIDG)

Wskazówki

  • Type Into do wpisania kryteriów wyszukiwania (Miejscowość + Ulica)
  • Check/Uncheck — zaznaczenie opcji wyszukiwania
  • Delay 2s po kliknięciu "Szukaj" — odczekaj na załadowanie wyników
  • Extract Table Data z ustawieniami: LimitExtractionTo: "Rows", MaximumResults: 50
  • Filter Data Table w trybie Remove — usuń wiersze gdzie REGON jest puste OR NIP jest puste
  • Sort Data Table po "Firma" rosnąco
Uwaga: W Filter Data Table łatwo pomylić tryb Keep z Remove. Czytaj warunek na głos: "USUŃ wiersze, w których REGON jest puste".

Folder: Dzien2_Zadanie2_WysylkaMailaZalacznik/ · Temat: SMTP + dynamiczna treść + załącznik

Zadanie 3 — Masowa wysyłka maili
Folder: Dzien2_Zadanie3_MasowaWysylkaMaili/ · Temat: Pętla + warunek + spersonalizowane emaile

Opis zadania

  1. Wczytaj plik Koszulki_email.xlsx (arkusz: Arkusz1) z danymi zamówień
  2. Pobierz dane logowania: Get Credential — AssetName: Robotivity_Credentials, FolderPath: Robotivity2
  3. Dla każdego wiersza (For Each Row in Data Table):
    • Wyciągnij dane: nazwa, rozmiar, kolor, status, data wysyłki, email odbiorcy
    • Warunek: str_status = "Zamówiono" Or str_status = "Zamowiono"
    • Jeśli tak — wczytaj treść z tresc_emaila_zamowiono.txt + String.Format
    • Jeśli nie — wczytaj treść z tresc_emaila_nie_zamowiono.txt + String.Format
    • Temat: "Status Twojego zamowienia: " + str_status
    • Wyślij przez Invoke Workflow File → Send_SMTP_Notification.xaml

Wskazówki

  • Read Range do wczytania pliku Excel z danymi
  • W pętli: Read Text File wczytuje szablon, String.Format wstawia dane
  • Warunek uwzględnia polskie znaki: "Zamówiono" Or "Zamowiono"
  • Wysyłka przez osobny workflow Send_SMTP_Notification.xaml z argumentami (in/out)
  • Argument out_str_Notification zwraca informację o wyniku wysyłki

Zadanie 4 — Pobieranie emaili i załączników
Folder: Dzien2_Zadanie4_PobieranieEmail_Zalaczniki/ · Temat: IMAP + filtrowanie + zapis załączników

Opis zadania

  1. Create Folder — stwórz folder Zalaczniki w katalogu projektu
  2. Pobierz dane logowania: Get Credential — AssetName: Robotivity_Credentials, FolderPath: Robotivity2
  3. Pobierz emaile z folderu "Robotivity" przez IMAP (max 30, NewestFirst)
  4. Dla każdego emaila (For Each — typ MailMessage):
    • Jeśli currentMailMessage.Attachments.Count > 0:
    • Zwiększ licznik, zaloguj nazwę załącznika
    • Save Email Attachments do folderu Zalaczniki
  5. Log Message: "Ilosc maili z zalacznikiem: " + int_counter.ToString
  6. Spakuj do ZIP: "Zalaczniki_" + Now.ToString("HH.mm.ss") + ".zip"

Wskazówki

  • Create Folder — Environment.CurrentDirectory + "\Zalaczniki"
  • Get IMAP Mail Messages — Server: imap.gmail.com, Port: 993, SSL: True, Folder: "Robotivity", Top: 30
  • For Each po liście emaili — filtr: Attachments.Count > 0
  • Save Email Attachments — ExcludeInlineAttachments: True
  • Compress/Zip Files do spakowania folderu z załącznikami

Zadanie 5 — Pobieranie maili do tabelki
Folder: Dzien2_Zadanie5_PobieranieMaili_Tabelka/ · Temat: IMAP + filtrowanie + DataTable + raport Excel

Opis zadania

  1. Pobierz dane logowania: Get Credential — AssetName: Robotivity_Credentials, FolderPath: Robotivity2
  2. Build Data Table z kolumnami: Numer_Zamowienia, Data_Zamowienia, Email
  3. Pobierz emaile z folderu "Robotivity" przez IMAP (max 30)
  4. Dla każdego emaila (For Each — typ MailMessage):
    • Jeśli temat zawiera "Zamówienie":
    • Numer zamówienia: currentMailMessage.Subject.Split(" ").Last
    • Data: currentMailMessage.DateAsDateTime.ToString("dd.MM.yyyy")
    • Email: currentMailMessage.Sender.Address
    • Add Data Row — dodaj wiersz do dt_raport
  5. Jeśli dt_raport.Rows.Count > 0 → Write Range do Zamowienia.xlsx (arkusz: Zamowienia)

Wskazówki

  • Build Data Table z kolumnami: Numer_Zamowienia, Data_Zamowienia, Email
  • .Subject.Split(" ").Last — wyciągnięcie numeru zamówienia z tematu
  • .DateAsDateTime.ToString("dd.MM.yyyy") — formatowanie daty (nie .Date!)
  • .Sender.Address — adres nadawcy (nie .From.Address!)
  • Add Data Row — ArrayRow: {str_numer_zamowienia, str_data_zamowienia, str_email_zamowienia}
  • If na końcu sprawdza czy tabela ma wiersze przed zapisem
