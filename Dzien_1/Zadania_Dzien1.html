Folder: Dzien1_Zadanie2_Kalkulator/ · Temat: Automatyzacja aplikacji desktopowej + dynamiczne selektory

Opis zadania

  1. Pobierz od użytkownika dwie liczby z przedziału 1–9

Zadanie 3 — Dni tygodnia
Folder: Dzien1_Zadanie3_DniTygodnia/ · Temat: Praca z plikiem tekstowym + tablica + pętla + warunek

Opis zadania

  1. Przeczytaj tekst z pliku Tydzien.txt
  2. Podziel pobrany tekst na poszczególne dni tygodnia
  3. Wypisz każdy dzień (używając Log Message) w pętli
  4. Jeżeli dniem tygodnia będzie "Piatek" — wyświetl Log Message: "Juz jutro weekend" i przerwij pętlę (Break)
  5. Jeśli dzień nie jest "Piatek" — wyświetl Log Message: "Dopiero [dzien] Index: [index]"

Wskazówki

  • Użyj Read Text File do odczytu pliku
  • Do podziału tekstu użyj metody .Split(";") na zmiennej typu String — wynik to tablica String[]
  • Do iteracji po tablicy użyj For Each (typ elementu: String)
  • Warunek sprawdzaj przez aktywność If
  • W gałęzi Then (Piatek): Log Message + aktywność Break — przerywa pętlę po znalezieniu piątku
  • W gałęzi Else: Log Message z informacją o bieżącym dniu i indeksie

Zawartość pliku Tydzien.txt

Poniedzialek;Wtorek;Sroda;Czwartek;Piatek;Sobota;Niedziela

Zadanie 4 — Kalkulator w Flowchart
Folder: Dzien1_Zadanie4_Flowchart_Kalkulator/ · Temat: Flowchart + FlowSwitch + FlowDecision

Opis zadania

  1. Korzystając z Flowcharta, stwórz prosty kalkulator
  2. Pobierz od użytkownika 2 liczby całkowite i wybrane działanie matematyczne (dodawanie; odejmowanie; mnozenie; dzielenie)
  3. Wykonaj obliczenia i wyświetl wynik w Message Box (sformatowany: dbl_wynik.ToString("F2"))
  4. FlowDecision: Jeśli wynik jest większy od 10 → Message Box: "Wynik wiekszy od 10"
  5. Jeśli wynik jest mniejszy lub równy 10 → Message Box: "Wynik mniejszy od 10"

Wskazówki

  • Utwórz nowy projekt jako Flowchart (nie Sequence)
  • Użyj FlowSwitch do rozgałęzienia na 4 działania matematyczne
  • Użyj FlowDecision do sprawdzenia warunku dbl_wynik > 10 — oba gałęzie (True i False) mają Message Box
  • Przy dzieleniu użyj zmiennej typu Double i konwersji cdbl(), aby wynik mógł być niecałkowity
  • Wynik formatuj z 2 miejscami po przecinku: dbl_wynik.ToString("F2")
Uwaga: Przy dzieleniu używaj cdbl() — bez konwersji: 7 / 2 = 3 (Integer obcina!). Z konwersją: cdbl(7) / cdbl(2) = 3.5.

Bonus Zadanie dodatkowe — Rozszerzony kalkulator
Temat: Rozszerzenie Zadania 4 o obsługę dzielenia przez zero i operację modulo

Opis zadania

  1. W przypadku dzielenia: dodaj FlowDecision — czy druga liczba = 0?

Ćwiczenie 5 — DataTable: Wprowadzenie
Folder: Dzien1_Zadanie5_DataTable_Wprowadzenie/ · Temat: Podstawowe operacje na DataTable (wspólnie z prowadzącym)

Opis ćwiczenia

  1. Utwórz DataTable z kolumnami: Imie, Nazwisko, Wiek i dodaj kilka wierszy
  2. Dodaj nową kolumnę do istniejącej tabeli
  3. Iteruj po wierszach pętlą For Each Row i loguj dane
  4. Usuń dodaną kolumnę
  5. Posortuj tabelę według kolumny Nazwisko
  6. Przefiltruj — zostaw tylko wiersze gdzie Wiek > 20
  7. Zapisz wynikową tabelę do pliku Excel

Kluczowe aktywności

Build Data Table  ·  Add Data Column  ·  Add Data Row  ·  For Each Row  ·  Remove Data Column  ·  Sort Data Table  ·  Filter Data Table  ·  Write Range

Opis zadania
